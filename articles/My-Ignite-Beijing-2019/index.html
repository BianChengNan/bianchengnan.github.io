Ignite Beijing 2019你参加了吗？多图，慎点！

发表于 2019-12-10 | 更新于: 2019-12-10 | 分类于 技术会议 | 热度： ℃
字数统计: | 阅读时长 ≈ 分钟

前言

第一次参加微软技术大会。之前对微软技术大会有太多遐想，今天终于亲身参与其中了。现在，终于有时间回顾下今天的 Ignite之旅了（全称是 Microsoft Ignite The Tour）。特记流水账一篇，以纪念我的第一次。

注册

11月份的时候听群友说，今年Microsoft Ignite北京站会在12月10，11号举办。马上注册。:sunglasses:

出发

昨晚睡觉前定好了今天早晨5:30的闹钟。闹钟响后，立刻起床。简单准备后，出门。扫了一辆小蓝骑到地铁站，然后坐到天津站赶往北京南站。到站后，把改好的变更集整理好发给客户。

北京南站——国家会议中心

坐地铁从北京南站到国家会议中心。4号线北京南站上车，到宣武门换乘2号线，到鼓楼大街站换乘8号线 ，在奥林匹克公园下车（更快的路线应该是4号线到平安里换乘6号线，然后到南锣古巷换乘8号线，在奥林匹克公园下车）。

2号线依然是老样子，人是真的多！我是被挤上车的，一点都不夸张。大家看看4号线换2号线的走廊，人山人海！

Ignite外景

下地铁后，从E口出来后，过马路，很容易找到微软大会的入口。

大家感受下：求知（Learn），求同（Connect），求索（Explore）

登记

进入大厅后，第一件事是登记。根据注册信息和能证明自己身份的证件（身份证，或者其它证件）拿到一张通行证，进入每个会场都需要这张卡。滴~

初印象

进入大厅后，先围绕一楼大厅转了一圈，发现有个展位在发小礼物，有很多人在排队，我就先去逛别处了，回来发现礼物没有了，应该是发完了，罢了。

早餐

有个初步印象后，当然要解决温饱问题了。根据右手边的指示牌+服务人员的指引，顺利来到就餐处。

预定特殊餐食指示牌再往里走就到了普通餐食区了。用餐需要先领“餐票”。

早餐是用袋子装起来的。

比较西式。

还可以自己接咖啡或者茶水。

快速吃完，开启今天的Ignite之旅。先看下指示图：

两侧是就餐的地方（我都是在右边吃的），靠近就餐区内侧的三个角是三个剧院式演讲区（Theater），另外一个角是微软展示区域（Microsoft ShowCase），再向里，分别有 创新企业展示区（StartUp Gallery），休息室（Feature Lounge），工作休息室（Work Lounge），娱乐休闲室（Play Lounge）。工作休息室提供了插座，可以用来给手机和笔记本充电，娱乐休闲室提供了唱吧，可以K歌。后面有安静区（Quiet Zone）和赞助商休息区（Sponsers Lounge），然后是走廊，走廊对面两侧分别设有男女洗手间。

来一张Work Lounge的特写，其它区域拍的不好，就不上图了。

Microsoft Team

在微软展示区域，最有意思的就是Microsoft Team的展示了。集成了在线聊天，视频会议，文件共享，多人同时文档编辑等诸多功能。看演示，效果很赞，不过我的手机拍照太差劲，屏幕太亮，录出来的视频没法看，太遗憾了。听现场的微软员工介绍，这里展示的是一体机（我看贴的是Surface的牌儿），系统是专门为Microsoft Team定制的（猜测应该是win10裁剪的:joy:）。一套下来，大概要9000$左右，根据硬件不同，价格有浮动。豪不豪？不过想买在国内也买不到:joy:，因为国内没有卖的。

放一张效果图吧。这效果，是不是该换手机了？

上午参会

在大厅逛了很久，心想，都在这了吗？不应该啊。到入口处才发现，还有分会场。（眼有点大:joy:）

在各个分会场逛了逛，参加了几场AZure相关的会议。拍照比较少，这里就不发图了。

午餐

逛了一上午，累了，该吃午餐了。和早餐流程一样，领一张“餐票”，拿着餐票领午餐。

下午参会

参加了一场AZure开发工具相关的会议，再次感叹vs code的强大。没有什么事是vs code插件解决不了的，如果有，就写一个。由于有事提前离场了，不知道演讲者有没有介绍Mark Russinovich大神开发的Sysinternals Suite for Nano Server相关工具。Sysinternals系列可是鼎鼎有名啊！

最后一场会议是在Theater1的老外讲的。主题是Windows Server 2019 的五大新功能。虽然只有短短的十分钟，但是整个过程声音洪亮，思路清晰，有声有色。感叹老外是真的自信。

晚餐

简单吃了几块糕点，喝了一杯橙汁。

惊喜

没想到又看到早晨发礼物的那家公司在发礼物，下（犹）定（犹）决（豫）心（豫）上前跟小姐姐要了一个红色小蘑菇（当然需要先扫码关注一下她们公司啦）。闺女的礼物有了。:sunglasses:

P.S. 到我闺女手里后，发现什么没？是不是孩子都这样啊？:rofl:

回家

早晨没来的及拍照，离行前，补发一张签到处的照片。签到处一共有两侧，另外一侧的工作人员还在忙。辛苦！

从里面出来后，月亮已经升起来了。

早晨来的时候，居然没发现这个标志性的丝路金桥。拍出来效果比较亮，实际天已经暗了。

拍完照，打道回府。虽然明天还有一天，但是没有非常感兴趣的主题，工作也比较忙。只能下次了。

再见了Ignite！再见了，北京！

福利

参加本次Ignite的小伙伴儿，都可以扫码注册Microsoft Learn，免费学习一门课程，有微软认证的哦。

总结

  • 本次大会很多会议是关于云的，能看出来微软对云生态的重视。
  • 与会的整个过程，服务人员很专业，为他们点赞。
  • 真心意识到良好表达能力的重要性！一定要增强自己在公共场合的表达能力。
  • 吐个槽：能不能把演讲者也显示出来？方便“追星族”追一下？:sunglasses:
