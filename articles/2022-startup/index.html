回首 2021

2021 年真是又忙又累的一年，一直在加班，清明加，五一加，周末更别提了。一直忙到年底，才真正休息了几天。2021 真的是在忙碌中度过的，好在努力没有白费，付出了不少，也收获了很多：

  • 交付的项目得到了客户的认可。
  • 我们的团队在不断壮大成长。
  • 我们的工作流程在持续优化中。
  • 我们的技术交流依旧在持续举办中。
  • 我们开始 code review 啦。
  • 我们开始全面拥抱 git 啦。

而且，年初立的几个 flag 算是基本实现了。

  1. 至少更新 12 篇原创，争取做到每月一篇。
  2. 争取再拿一篇看雪精华，至少一篇优秀。
  3. 写一篇关于系统优化的实战。目前还没有写一篇真正让自己满意的优化文章。
  4. 坚持分享更多书籍，至少 12 本。

其中，最后一篇文章获得了看雪精华，真是太太太开心了，比高阅读量还要开心。

说明： 公众号 【编程难】 是我的个人公众号，【比目信息】是公司公众号， 2021 的原创基本上都发布在公司公众号上了，每隔一段时间会转载到个人公众号上。

展望 2022

在忙碌中度过了 2021，希望 2022 能更上一层楼。

  • 2022 年，首要任务是锻炼身体。身体是革命的本钱。

  • 加深团队成员对 git 的理解。

  • 继续坚持 code review，增强代码质量。

  • 继续坚持技术交流，不断提升团队成员的技能。

  • 继续坚持分享技术文章，分享至少 12 篇原创。如果能多获得几次看雪精华就太好了。
